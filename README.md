# 🛒 E-Commerce Web Application

## 🚀 Overview
This is a full-stack e-commerce web application built using React and Spring Boot. It allows users to browse products, add items to cart, and place orders securely.

---

## 🔥 Features
- User authentication using JWT  
- Product listing and search  
- Add to cart and checkout  
- Order management system  
- RESTful APIs (15+ endpoints)  

---

## 🛠 Tech Stack
- Frontend: React.js, HTML, CSS  
- Backend: Spring Boot  
- Database: MySQL  
- Authentication: JWT  

---

## ▶️ How to Run

### Backend
- Open backend folder  
- Run Spring Boot application  

### Frontend
- Open frontend folder  
- Run:
```bash
npm install
npm start
